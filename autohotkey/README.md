# Hungarian characters for AutoHotkey

The [us_hu.ahk](us_hu.ahk) file can be used with AutoHotkey to input Hungarian characters and the
Euro sign on any keyboard layout.

## The characters

| Key combination | Character |
| --------------- | :-------: |
| Right Alt + A   | á |
| Right Alt + E   | é |
| Right Alt + I   | í |
| Right Alt + O   | ó |
| Right Alt + P   | ö |
| Right Alt + [   | ő |
| Right Alt + U   | ú |
| Right Alt + Y   | ü |
| Right Alt + T   | ű |
| Right Alt + 5   | € |
